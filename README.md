# 📌 Maincrafts Internship – Task 4
## Task Manager Dashboard (CRUD Application)

This project is developed as part of the **Full Stack Web Development Internship – Task 4** at Maincrafts.

## 🧠 Task Objective
To build a **Task Manager Dashboard** using frontend technologies that performs CRUD operations and stores data using LocalStorage.

---

## 🚀 Features Implemented
- ✅ Add new tasks
- ✅ Display all tasks dynamically
- ✅ Edit existing tasks
- ✅ Delete tasks
- ✅ Search/filter tasks
- ✅ Data persistence using LocalStorage
- ✅ Responsive and clean UI design

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox for layout)
- JavaScript (DOM Manipulation & LocalStorage)

---

## ⚙️ How It Works
- Tasks are stored in browser LocalStorage
- JavaScript dynamically updates UI without page reload
- Search filters tasks in real-time

---

## ▶️ How to Run
1. Clone or download the repository
2. Open `index.html` in any browser

---

## 🎯 Learning Outcomes
- Implemented CRUD operations using JavaScript
- Learned LocalStorage handling
- Improved UI design using CSS Flexbox
- Built a dynamic and interactive web application
